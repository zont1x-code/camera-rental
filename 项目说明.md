# Camera Rental 项目说明

## 项目概述

相机租赁预约网站，前后端一体化。技术栈：

| 层 | 技术 |
|----|------|
| 前端 | 原生 HTML/CSS/JS，Apple 风格暗黑主题 |
| 后端 | Node.js + Express |
| 数据库 | MySQL 8（本地）/ TiDB Cloud（生产） |
| 文件存储 | 本地 uploads/ 目录 |

---

## 快速启动（本地）

### 1. 安装依赖
```bash
npm install
```

### 2. 创建数据库
```bash
mysql -u root -p -e "CREATE DATABASE camera_rental CHARACTER SET utf8mb4"
```

### 3. 配置 .env
```
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=camera_rental
```

### 4. 初始化表结构 + 导入数据
```bash
node seed.js
```

### 5. 启动
```bash
npm start
```

访问 `http://localhost:3000`

## 部署上线

### 架构
```
[用户] → [Render (Node.js)] → [TiDB Cloud (MySQL)]
              ↓
    GitHub (代码仓库)
```

### 部署步骤

**1. 推送代码到 GitHub**
```bash
git init
git add .
git commit -m "init"
git remote add origin https://github.com/你的用户名/camera-rental.git
git push -u origin main
```

**2. Render 部署**
- Render 新建 Web Service → 连接 GitHub 仓库
- Build Command: `npm install`
- Start Command: `npm start`
- 环境变量：
  ```
  DB_HOST=gateway01.xxx.tidbcloud.com
  DB_PORT=4000
  DB_USER=你的TiDB用户
  DB_PASSWORD=你的TiDB密码
  DB_NAME=camera_rental
  ```

**3. TiDB 数据库**
- TiDB Cloud 创建免费集群 → 获取连接信息
- 用 Navicat 连接 TiDB → 运行 `schema.sql` 建表
- 把 TiDB 的连接信息填到 Render 环境变量

**4. 域名**
- Render 自动分配 `xxx.onrender.com`

### 安全注意事项（生产上线前必须）

- `admin_users` 表中密码是明文存储，生产环境需要用 bcrypt 加密
- `.env` 不上传 GitHub，通过 Render 环境变量管理
- API Key（DeepSeek）不要硬编码
